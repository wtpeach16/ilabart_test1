# 사진 폴더 안내

이 폴더에 아이랩아트 실제 사진을 넣어주세요.

## 권장 파일명

| 파일명 | 용도 | 권장 크기 |
|--------|------|-----------|
| `hero.jpg` | 메인 히어로 배경 | 1920×1080 |
| `studio-1.jpg` | 아틀리에 내부 | 1200×800 |
| `studio-2.jpg` | 수업 공간 | 1200×800 |
| `class-1.jpg` ~ `class-6.jpg` | 갤러리 (수업 사진) | 800×800 |
| `logo.png` | 로고 (PNG 버전, 선택) | 400×400 |

## 사진 추가 방법

1. 위 파일명으로 사진을 이 폴더에 복사
2. GitHub에 push하면 홈페이지에 반영됩니다
3. 사진이 없으면 CSS placeholder가 표시됩니다
